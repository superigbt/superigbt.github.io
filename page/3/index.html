绍兴晶桥

发表于 2022-11-03 更新于 2022-11-13 分类于 绍兴晶桥
客户要求

  1. 用在搅磨机上做供电用，输入AC电压85~264V，输出电压分为两路，一路为15V带风扇0.5A左右，另外一路也是15V带驱动电路100mA就可以，总功率10W左右。整机工作一般在5小时左右，IC温升控制在85℃以下。PCB板的宽度要求是30mm，长度与高度暂时没有要求。 实际产品出售在国内，220VAC为常用电压。

  2. 客户要求提供电路图变压器参数以及Demo板。

DEMO板
  1. 2022年10月11号，寄出REF7276S 单路输出12V 800mA的DEMO板给客户测试。
  2. 2022年10月25号，客户测试Demo板满意，原理图己给客户，工程师评板是否可以layout的下。
客户定板

最终客户选择12V 400+的电流，单路输出，做个小板子给 32mm*21mm，只给风扇独立供电。2022年11月3号己经layout好，并发给谭工审核，并协助计算变压器

科领表计REF8410替换MP2451

发表于 2022-10-26 更新于 2022-12-11 分类于 宝惠丰

客户主要做电表板卡，产品销往尼日利亚，印度，东南亚等。 ACDC采用MP175,DCDC采用MP2451，基中REF8410替换客户的MP2451，但是ACDC暂时客户没有更换的意愿。

2022/12/08

客户在竟标非洲电表项目。DCDC采用REF8410

2022/10/16

客户试产REF8410一盘。

2022/07/13

样机目前没有发现问题，还在老化中。现在了解老方案，给工程师推荐REF7001D

2022/06/23

科领试生产30台样机，申请了50个样品

2020/06/10，

客户ACDC采用阻容降压，岳百DCDC采用REF8410, 220VAX 50HZ，会有一个100KHz的4000V的干扰，会引起电路保护。谭工沟通后，认为与DCDC关系不大，串一个电感可以解决。但是串的电感 压敏前与压敏后的效果不一样。

2022/05/26

客户的工程师了解到了REF7001D，但是他们只在老方案上采用过TNY27X，宝惠丰吴经理现在还未了到解到客户是如保得知REF7001D的，但代理商反馈客户测试REF8410初步可用。

2022/05/18

客户电源芯片采用MP174与MP2451,代理商沟通替换。只有REF8410替换会容易一下。MP174如果要换需要改板，只能在新的方案上看是否有机会
